C Sharp

O C# é uma linguagem de programação, multiparadigma, de tipagem forte criada em conjunto com a arquitetura da plataforma .NET da Microsoft. A linguagem C# foi influenciada por diversas linguagens diferentes, como JAVA e C++, por exemplo. A linguagem foi construída em cima das principais vantagens das linguagens mencionadas acima, melhorando apenas as suas implementações e adicionando novos recursos, fazendo a linguagem atrativa para desenvolvedores que queiram migrar para o Microsoft .NET.

Sobre a Linguagem

C#, assim como linguagens como o Java, é uma linguagem orientada a objetos. Por consequência, ela dá suporte ao uso de conceitos como encapsulamento, herança e polimorfismo na construção de projetos utilizam que utilizam a linguagem como base, um exemplo mais comum da aplicação desses conceitos, está no fato de que toda variável e método declarado em um código C# é encapsulado dentro das definições de uma classe no qual são declarados.

O C#, foi a primeira linguagem orientada a objetos da família C/C++. Ela segue os padrões da POO, onde tudo deriva de um ancestral comum, que no caso do C# é a superclasse, System.Object. Além de ser uma linguagem orientada a objetos, o C# também é uma linguagem fortemente tipada que possui suporte a DLL's, COM e COM+, é case-sensitive e suas classes podem implementar várias interfaces, embora a herança por extensão é simples. Seus programas executam sobre um ambiente gerenciável, ficando a cargo do .NET framework realizar o controle de memória. Possui a Language Integrated Query (LINQ), que fornece recursos de consulta interna entre uma variedade de fontes de dados.

Entendendo o .NET Framework

O ".NET" é uma plataforma ou framework de desenvolvimento open-source criado pela Microsoft, e que é composto por uma biblioteca, um compilador e uma máquina virtual. Dentro da plataforma, esses componentes possibilitam a criação de códigos em linguagens como C#, VB.NET e F#. Como podemos ver, a plataforma é voltada para o uso de linguagens e tecnologias da Microsoft. O propósito desse framework é o desenvolvimento de uma plataforma que possibilite a criação de projetos para essas tecnologias, sem que seja necessário usar bibliotecas diferentes para cada uma delas.

Aplicações do .NET Framework

Na primeira versão da plataforma, que foi o .NET Framework, só era possível desenvolver aplicações desktop para Windows, aplicações web com o framework Asp.NET para serem hospedadas em servidores com Windows Server e aplicações mobile para Windows Phone. Depois que a Microsoft abraçou o open-source, surgiu a possibilidade de criar aplicativos para desktop, web, mobile utilizando o framewok Xamarim Forms, smart tvs, vídeo games e até mesmo em embarcados, como o Arduino para projetos de internet das coisas ou automação, além de que também roda no Windows, distribuições Linux e Mac OSX.
